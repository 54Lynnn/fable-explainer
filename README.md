# Fable Explainer 🧚

> 用寓言故事 + 技术解释的方式让复杂概念一听就懂。

当你说"用寓言解释XXX"，它不只是给你念定义——**先用故事让你记住画面，再揭晓概念**。

## 🎯 三种教学模式

### 寓言故事（默认）
创造贴合概念的全新寓言，故事走向精确映射概念的核心机制。结尾揭晓 + 技术解释。

### 日常类比
把概念映射到日常熟悉的场景，不打故事直接打比方。

### 层层递进
同一概念从"一句话版"到"入门版"到"深入版"，逐步加深。

## 🚀 用法

> "用寓言解释什么是过拟合"
> "给我打个比方，说说区块链"
> "从头到尾讲一遍分布式系统"
