# Fable Explainer 🧚

> 用寓言故事 + 技术解释的方式让复杂概念一听就懂。

基于 OpenClaw / Hermes 的 AI 技能。当你说"用寓言解释XX"，它会用一个生动的故事帮你理解，再附上技术层面的说明。

[![ClawHub](https://img.shields.io/badge/ClawHub-fable--explainer-00A1D6)](https://clawhub.ai/54lynnn/fable-explainer)
[![GitHub](https://img.shields.io/badge/GitHub-54Lynnn%2Ffable--explainer-181717)](https://github.com/54Lynnn/fable-explainer)

## 安装

```bash
clawhub install fable-explainer
```

## 用法

> "用寓言解释什么是过拟合"
> "用故事帮我理解区块链"
